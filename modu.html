Oque é a Modelização?

Em programação, molduração (ou padding, em inglês) é o processo de adicionar espaços em branco ou caracteres especiais antes ou depois de uma string ou outro tipo de dado, para que ele atinja um determinado tamanho ou formato. Isso é feito para facilitar a apresentação ou formatação dos dados de uma maneira específica. Por exemplo, ao exibir um número em uma tabela, é possível preencher os espaços à esquerda com zeros para que todos os números tenham o mesmo número de dígitos. Da mesma forma, é possível adicionar espaços em branco à direita de uma string para que todas as strings em uma lista tenham o mesmo comprimento. A modulação é uma técnica amplamente utilizada em programação para formatação de saída de dados, especialmente em programação de interfaces gráficas, edição de texto e impressão de relatórios.
